Create a TIP admin prototype

Design iterations

Version 3 (current)

  1. My account page: Edit wording for links for a completed TIP (no reference to TIP report), instead showing links to PDF, HTML and XLS
  2. My account page: Updated and added new status 'Pending Approval' for application
  3. Task list page: Remove the "in progress" TIP or to-do TIP from the task list
  4. Task list page: Make a decision on the action word for the task list (Respond for priority / Review for other)
  5. Task list page: Streamline the approval process so that back-end approval finalises the submission, avoiding an extra confirmation step:
    1. Record TIP actions
    2. Review your draft TIP. (Moving confirmation step to Section 3)
    3. Submit your TIP .
  6. Priority recommendations - with filter: Implement dynamic column widths for risk / recommendation to accommodate longer text entries
  7. Priority recommendations - with filter: Add an "in progress" status option to the Status column to allow users to save incomplete sections and return later, including filters
  8. Respond to Priority recommendation and action: Removed first question and retain the 'Do you want to add an action' question only
  9. Respond to Priority recommendation and action: Added addittional 'I will respond later' option for each question and fields
  10. Respond to Priority recommendation and action: Added logic, if at least one 'I will respond later' option is selected the task cannot be completed, and it must display 'In progress' on the filtered view
  11. Respond to Other recommendations: Implemented a cleaner pattern by including 'Save and return as in progress' button. If user is using this button, we can skip the validations for the form.
    (No logic in place, design pattern only)
  12. Check actions page: Remove 'yes' from 'no action added' and show 'No action' with reason in the same row
  13. Check actions page: Add 'Review recommendations' above the Action details row
  14. Confirmation page: Create clearer confirmation messaging and email notification for applicants once their submission is approved
  15. Email confirmation via Gov Notify: Outstanding
    Create automated e-mail notification for applicants once their submission is approved, enabling them to access and download their final report
